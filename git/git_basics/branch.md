# Git branch

## 1. commands

* 브랜치 생성
  * ```text
     (master) $ git branch {브랜치명}
    ```
* 브랜치 이동
  * ```text
     (master) $ git checkout {브랜치명}
    ```
* 브랜치 생성 및 이동
  * ```text
     (master) $ git checkout -b {브랜치명}
    ```
* 브랜치 삭제
  * ```python
     (master) $ git branch -d {브랜치명}
    ```
* 브랜치 목록
  * ```text
     (master) $ git branch
    ```
* 브랜치 병합
  * ```text
     (master) $ git merge {브랜치명}
    ```
